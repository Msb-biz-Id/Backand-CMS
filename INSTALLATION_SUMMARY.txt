🚀 QUICK START (5 MINUTES)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  
  FOR WINDOWS/LARAGON:
  
  1. Copy workspace/ to: C:\laragon\www\advancedcms\
  
  2. Create Virtual Host in Laragon:
     Name: advancedcms.test
     Path: C:\laragon\www\advancedcms\public
  
  3. Create Database (HeidiSQL):
     CREATE DATABASE cms_database;
  
  4. Configure .env:
     copy .env.example .env
     Edit: DB_NAME=cms_database
  
  5. Install Database (Laragon Terminal):
     php database\install.php
  
  6. Access CMS:
     http://advancedcms.test/admin
     Email: admin@example.com
     Password: admin123
